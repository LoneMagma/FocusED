## Installation

1. Download the APK above
2. **Settings → Security → Install unknown apps** → allow your browser/file manager
3. Tap the APK to install
4. Open FocusED → **Enable Service** → grant Accessibility permission

---

## Permissions

- `BIND_ACCESSIBILITY_SERVICE` — watches for Reels/Shorts UI, sends back press
- No internet. No location. No contacts. Nothing else.

---

## Requirements

- Android 8.0 (Oreo) or higher
- ~200 KB storage
